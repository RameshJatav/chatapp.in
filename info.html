💬 Real-Time Chat App Project

This is a full-stack real-time chat application. Users register with a mobile number and password, log in securely, and can add contacts. Once both users accept the connection, they can securely chat in real-time with optional image sharing.

  • Registration/Login using mobile number & password
  • JWT authentication for secure sessions
  • Real-time messaging using Socket.IO
  • Image sharing with upload preview
  • Mobile-first responsive layout
